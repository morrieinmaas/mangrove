# v0.3.0 Plan 1 — Lossless CST Foundation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a lossless concrete-syntax tree (CST) for Mangrove inside the
`mangrove-syntax` crate, with a lowering pass that reproduces the existing
`Document`/`Value` AST byte-for-byte, so the formatter and LSP (later plans) have a
comment-and-trivia-preserving tree to walk.

**Architecture:** A new lossless lexer emits *every* byte (whitespace, comments,
newlines, tokens) with byte ranges. An event-emitting recursive-descent parser builds a
`rowan` green tree mirroring the existing parser's grammar. A `lower` pass walks the CST
and produces the existing `Document`. The existing hand-written parser is kept as an
**equivalence oracle**: for every fixture, `hash(lower(parse_cst(src)))` must equal
`hash(parse(src))`. Only after the whole corpus passes do we flip the public `parse()` to
route through the CST.

**Tech Stack:** Rust 2024, `rowan` (lossless syntax trees, the rust-analyzer model),
existing `mangrove-core` (`Value`, `StrPart`), `num-bigint`, `bigdecimal`.

## Why this is Plan 1 of 5 (v0.3.0 decomposition)

v0.3.0 = five sequential sub-plans, all shipping under the v0.3.0 tag:
1. **CST foundation** (this plan) — unblocks everything.
2. **`mangrove fmt`** — walks the CST; written once this plan's CST API is concrete.
3. **LSP server** — walks the CST + reuses typed/compose diagnostics.
4. **Editor integration** — Neovim plugin + Zed extension.
5. **k8s migration docs + example fix** — (the `gen-openapi` constraint-mapping piece is
   already shipped on `main`).

Plans 2–5 are written *after* this lands, because their concrete code depends on the CST
node API defined here.

```rust
/// `rowan::SyntaxKind(u16)`. Order is irrelevant except `__LAST` must be last.
#[derive(Debug, Clone, Copy, PartialEq, Eq, PartialOrd, Ord, Hash)]

pub enum SyntaxKind {

    WHITESPACE, // spaces, tabs, CR
    COMMENT,    // `# ...` ordinary comment
    // ---- significant tokens (mirror lexer::Tok) ----
    NEWLINE,
    L_BRACE, R_BRACE, L_BRACKET, R_BRACKET, L_PAREN, R_PAREN,

    INT, DECIMAL, UNIT_LIT, STR, INTERP_STR, BOOL, BYTES,
    BAREWORD, DOC, DIRECTIVE,
    ERROR,  // an unrecognized/!invalid token (used by recovery, Task 11)
    EOF,
    // ---- nodes ----
    DOCUMENT,
    UNIT_DEF, TYPE_DEF, SCHEMA_DECL, USE_DECL, PARAM_DECL, FN_DEF,
    // values / body
    RECORD, FIELD, LIST, LIST_OP_ITEM, MATCH_EXPR, MATCH_ARM, CALL, SPREAD,
    UNSET, REF, INTERP, ANNOTATION, DEFAULT, BINDING,

    TYPE_MAP, TYPE_LIST, TYPE_UNION, TYPE_NAMED, TYPE_BRAND, REQUIRE, PRED,
    __LAST,
}

        matches!(self, SyntaxKind::WHITESPACE | SyntaxKind::COMMENT)
    }
}

/// The rowan `Language` marker for Mangrove.

impl rowan::Language for MangroveLang {
    type Kind = SyntaxKind;

        assert!(raw.0 < SyntaxKind::__LAST as u16, "bad SyntaxKind {}", raw.0);

        // repr(u16) guarantees the discriminant layout, and the assert bounds it.
        unsafe { std::mem::transmute::<u16, SyntaxKind>(raw.0) }

    fn kind_to_raw(kind: Self::Kind) -> rowan::SyntaxKind {
        rowan::SyntaxKind(kind as u16)
    }
}

pub type SyntaxToken = rowan::SyntaxToken<MangroveLang>;
```

- [ ] **Step 3: Implement `lex_lossless`.** Operate on **bytes via char indices** to keep
  byte offsets correct for multi-byte UTF-8. Reuse the *recognition rules* from
  `super::super::lexer` (same token shapes) but: (a) emit `WHITESPACE` and `COMMENT` tokens
  instead of skipping them, (b) record byte `start`/`end`, (c) never fail — an
  unrecognized byte becomes a single-byte `ERROR` token (kept lossless). Write to
  `cst/lex.rs`:

```rust
//! Lossless lexer: every byte of the source lands in exactly one token, so the
//! token texts concatenate back to the original. Trivia (whitespace, comments)
//! are emitted, not discarded. Never errors — unknown bytes become `ERROR`.

use super::kind::SyntaxKind;

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub struct LosslessTok {
    pub kind: SyntaxKind,
    pub start: usize,
    pub end: usize,
}

pub fn lex_lossless(src: &str) -> Vec<LosslessTok> {
    let bytes = src.as_bytes();
    let mut out = Vec::new();
    let mut i = 0usize;
    while i < bytes.len() {
        let start = i;
        let kind = match bytes[i] {
            b' ' | b'\t' | b'\r' => {
                while i < bytes.len() && matches!(bytes[i], b' ' | b'\t' | b'\r') { i += 1; }
                SyntaxKind::WHITESPACE
            }
            b'\n' => { i += 1; SyntaxKind::NEWLINE }
            b'#' => {
                // `##` doc, `#!` directive, else ordinary comment — all run to EOL.
                let kind = match bytes.get(i + 1) {
                    Some(b'#') => SyntaxKind::DOC,
                    Some(b'!') => SyntaxKind::DIRECTIVE,
                    _ => SyntaxKind::COMMENT,
                };
                while i < bytes.len() && bytes[i] != b'\n' { i += 1; }
                kind
            }
            _ => {
                // Delegate one significant token to a helper that mirrors the eval
                // lexer's rules and returns (kind, new_index). See Step 4.
                let (k, next) = super::lex::scan_significant(src, i);
                i = next;
                k
            }
        };
        out.push(LosslessTok { kind, start, end: i });
    }
    out.push(LosslessTok { kind: SyntaxKind::EOF, start: bytes.len(), end: bytes.len() });
    out
}
```

- [ ] **Step 1: Define the parser driver.** The driver holds the lossless token list and a
  cursor, and a `GreenNodeBuilder`. **Trivia handling:** the builder attaches each
  whitespace/comment/newline token to the tree as it advances, so the tree stays lossless;
  significant-token consumption skips trivia by feeding it to the builder first. Write to
  `cst/parse.rs`:

```rust
//! Event-emitting recursive-descent parser that builds a lossless `rowan` green
//! tree. Mirrors the grammar of `super::super::parser` but emits nodes/tokens
//! instead of constructing AST values. Lowering (`super::lower`) turns the tree
//! back into the existing `Document`.

use super::kind::{MangroveLang, SyntaxKind, SyntaxNode};
use super::lex::{lex_lossless, LosslessTok};
use super::super::parser::ParseError;
use rowan::GreenNodeBuilder;

pub struct Parse {
    pub green: rowan::GreenNode,
    pub errors: Vec<ParseError>,
}

impl Parse {
    pub fn syntax(&self) -> SyntaxNode {
        SyntaxNode::new_root(self.green.clone())
    }
}

struct Parser<'a> {
    src: &'a str,
    toks: Vec<LosslessTok>,
    pos: usize,
    builder: GreenNodeBuilder<'static>,
    errors: Vec<ParseError>,
}

impl<'a> Parser<'a> {
    fn new(src: &'a str) -> Self {
        Parser { src, toks: lex_lossless(src), pos: 0, builder: GreenNodeBuilder::new(), errors: Vec::new() }
    }

    /// The kind of the next *significant* token (trivia skipped, not consumed).
    fn current(&self) -> SyntaxKind {
        let mut i = self.pos;
        while i < self.toks.len() && self.toks[i].kind.is_trivia() {
            i += 1;
        }
        self.toks[i].kind
    }

    /// Push all pending trivia onto the tree, then return.
    fn eat_trivia(&mut self) {
        while self.pos < self.toks.len() && self.toks[self.pos].kind.is_trivia() {
            self.push_token();
        }
    }

    /// Emit the current raw token (any kind) into the builder and advance.
    fn push_token(&mut self) {
        let t = self.toks[self.pos];
        let text = &self.src[t.start..t.end];
        self.builder.token(rowan_kind(t.kind), text);
        self.pos += 1;
    }

    /// Consume one significant token (after attaching leading trivia).
    fn bump(&mut self) {
        self.eat_trivia();
        if self.toks[self.pos].kind != SyntaxKind::EOF {
            self.push_token();
        }
    }

    fn start(&mut self, kind: SyntaxKind) {
        self.eat_trivia();
        self.builder.start_node(rowan_kind(kind));
    }
    fn finish(&mut self) {
        self.builder.finish_node();
    }
}

fn rowan_kind(k: SyntaxKind) -> rowan::SyntaxKind {
    use rowan::Language;
    MangroveLang::kind_to_raw(k)
}

pub fn parse_cst(src: &str) -> Parse {
    let mut p = Parser::new(src);
    p.start(SyntaxKind::DOCUMENT);
    parse_document(&mut p);
    p.eat_trivia(); // trailing trivia before EOF stays in the tree
    p.finish();
    Parse { green: p.builder.finish(), errors: p.errors }
}

fn parse_document(p: &mut Parser) {
    // Filled out construct-by-construct in Task 6+. For Task 4 we handle only the
    // simplest binding: `bareword : value` where value is an int/str/bool. This
    // proves the event→tree→lower loop end to end before porting the full grammar.
    while p.current() != SyntaxKind::EOF {
        if p.current() == SyntaxKind::NEWLINE {
            p.bump();
            continue;
        }
        parse_binding(p);
    }
}

fn parse_binding(p: &mut Parser) {
    p.start(SyntaxKind::BINDING);
    p.bump(); // key (BAREWORD or STR)
    if p.current() == SyntaxKind::COLON { p.bump(); }
    parse_atom(p);
    p.finish();
}

fn parse_atom(p: &mut Parser) {
    match p.current() {
        SyntaxKind::INT | SyntaxKind::STR | SyntaxKind::BOOL | SyntaxKind::DECIMAL
        | SyntaxKind::UNIT_LIT | SyntaxKind::INTERP_STR | SyntaxKind::BYTES => p.bump(),
        _ => {
            // Task 11 turns this into recovery; for now consume one token as ERROR-ish.
            p.bump();
        }
    }
}
```

```rust
//! Lower a lossless CST back into the existing `Document`/`Value` AST for
//! evaluation. Trivia is ignored here (it only matters to the formatter/LSP).

use super::kind::{SyntaxKind, SyntaxNode, SyntaxToken};
use super::super::parser::{Document, ParseError};
use rowan::NodeOrToken;

pub fn lower(node: &SyntaxNode) -> Result<Document, ParseError> {
    debug_assert_eq!(node.kind(), SyntaxKind::DOCUMENT);
    // For Task 5: build a Document whose body is a record of the simple bindings.
    // Extend construct-by-construct in Tasks 6+.
    let mut fields = Vec::new();
    for child in node.children() {
        if child.kind() == SyntaxKind::BINDING {
            fields.push(lower_binding(&child)?);
        }
    }
    Ok(Document::from_simple_fields(fields)) // helper added below
}

/// `(key, Value)` for one simple binding.
fn lower_binding(node: &SyntaxNode) -> Result<(String, mangrove_core::Value), ParseError> {
    let mut toks = node.children_with_tokens().filter_map(|e| match e {
        NodeOrToken::Token(t) if !t.kind().is_trivia() => Some(t),
        _ => None,
    });
    let key_tok = toks.next().expect("binding has a key");
    let key = decode_key(&key_tok);
    // skip COLON
    let val_tok = toks.find(|t| t.kind() != SyntaxKind::COLON).expect("binding has a value");
    let value = decode_scalar(&val_tok)?;
    Ok((key, value))
}

fn decode_key(t: &SyntaxToken) -> String {
    match t.kind() {
        SyntaxKind::STR => unescape_str(t.text()),
        _ => t.text().to_string(), // BAREWORD
    }
}

fn decode_scalar(t: &SyntaxToken) -> Result<mangrove_core::Value, ParseError> {
    use mangrove_core::Value;
    let text = t.text();
    Ok(match t.kind() {
        SyntaxKind::INT => Value::Int(text.parse().expect("lexer guaranteed int")),
        SyntaxKind::BOOL => Value::Bool(text == "true"),
        SyntaxKind::STR => Value::Str(unescape_str(text)),
        // DECIMAL/UNIT_LIT/INTERP_STR/BYTES handled in later tasks.
        other => unreachable!("decode_scalar not yet handling {other:?}"),
    })
}

/// Strip surrounding quotes and apply escapes — REUSE the legacy decoder rather
/// than reimplementing. If the legacy unescape is private in `lexer.rs`, make it
/// `pub(crate)` and call it here.
fn unescape_str(raw: &str) -> String {
    super::super::lexer::unescape_quoted(raw) // make pub(crate) in lexer.rs
}
```

```rust
    fn push_error(&mut self, msg: impl Into<String>) {
```
