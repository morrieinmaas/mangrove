# M1 — L0 Data Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:executing-plans (inline) or subagent-driven-development. Steps use `- [ ]` checkboxes. Design: `../specs/2026-06-24-m1-l0-data-design.md`.

**Goal:** `mangrove hash file.mang` parses an L0 document → canonical form → deterministic CBOR → BLAKE3 `b3:<hex>`, with conformance vectors pinning the bytes.

**Architecture:** Three new crates — `mangrove-syntax` (lexer+parser → `Value`), `mangrove-cbor` (hand-written deterministic encoder), `mangrove-canonical` (value → CBOR → hash). The `Value` model lives in `mangrove-core`.

**Tech Stack:** Rust 2024; `num-bigint`, `bigdecimal`, `blake3`; hand-written CBOR (no crate).

## Global Constraints

- Edition 2024, Apache-2.0, workspace inheritance, `unsafe_code = "forbid"`, clippy `-D warnings` (see roadmap §M0).
- Gate after every task: `just ci` green. Verify clippy via `rtk proxy cargo clippy …` locally (the hooked `cargo clippy` misreports — see memory).
- No IEEE float ever in the model (axiom §2.5). `Int` vs `Decimal` decided by surface syntax, never inference.
- Commit directly to `main` (trunk-based, this repo).

```rust
    Str(String),
    Bool(bool),
    Bytes(Vec<u8>),
    List(Vec<Value>),
    Map(BTreeMap<String, Value>),

    pub fn map_keys(&self) -> Vec<String> {

            Value::Map(m) => m.keys().cloned().collect(),
            _ => Vec::new(),

    }
}
```

```rust
//! Deterministic canonical CBOR encoder for `mangrove_core::Value`.
//!
//! Follows RFC 8949 §3 wire format and §4.2 determinism (shortest-form
//! head, definite lengths, no floats) — EXCEPT map keys are ordered by
//! Mangrove spec §7.1 (Unicode code point), not RFC 8949 length-first.

use bigdecimal::BigDecimal;
use mangrove_core::Value;
use num_bigint::BigInt;
use num_traits::{Signed, ToPrimitive};

pub fn encode(value: &Value) -> Vec<u8> {
    let mut out = Vec::new();
    encode_into(value, &mut out);
    out
}

fn encode_into(value: &Value, out: &mut Vec<u8>) {
    match value {
        Value::Int(n) => encode_int(n, out),
        Value::Decimal(d) => encode_decimal(d, out),
        Value::Str(s) => encode_str(s, out),
        Value::Bool(b) => out.push(if *b { 0xf5 } else { 0xf4 }),
        Value::Bytes(b) => {
            encode_head(2, b.len() as u64, out);
            out.extend_from_slice(b);
        }
        Value::List(xs) => {
            encode_head(4, xs.len() as u64, out);
            for x in xs {
                encode_into(x, out);
            }
        }
        Value::Map(m) => {
            encode_head(5, m.len() as u64, out);
            for (k, v) in m {
                encode_str(k, out);
                encode_into(v, out);
            }
        }
    }
}

/// Major type in the top 3 bits; argument in shortest of the 1/2/3/5/9-byte forms.
fn encode_head(major: u8, arg: u64, out: &mut Vec<u8>) {
    let mt = major << 5;
    if arg < 24 {
        out.push(mt | arg as u8);
    } else if arg <= u8::MAX as u64 {
        out.push(mt | 24);
        out.push(arg as u8);
    } else if arg <= u16::MAX as u64 {
        out.push(mt | 25);
        out.extend_from_slice(&(arg as u16).to_be_bytes());
    } else if arg <= u32::MAX as u64 {
        out.push(mt | 26);
        out.extend_from_slice(&(arg as u32).to_be_bytes());
    } else {
        out.push(mt | 27);
        out.extend_from_slice(&arg.to_be_bytes());
    }
}

fn encode_str(s: &str, out: &mut Vec<u8>) {
    encode_head(3, s.len() as u64, out);
    out.extend_from_slice(s.as_bytes());
}

fn encode_int(n: &BigInt, out: &mut Vec<u8>) {
    if !n.is_negative() {
        match n.to_u64() {
            Some(u) => encode_head(0, u, out),
            None => encode_bignum(6 << 5 | 2, n.magnitude_bytes(), out), // placeholder; see below
        }
    } else {
        let arg = (-n) - 1; // CBOR negint: value = -1 - arg
        match arg.to_u64() {
            Some(u) => encode_head(1, u, out),
            None => encode_bignum_tag(3, &arg, out),
        }
    }
}
```
Replace the positive-bignum arm and helpers with this correct version (do NOT keep the placeholder line):
```rust
fn encode_int(n: &BigInt, out: &mut Vec<u8>) {
    if !n.is_negative() {
        match n.to_u64() {
            Some(u) => encode_head(0, u, out),
            None => encode_bignum_tag(2, n, out),
        }
    } else {
        let arg = (-n) - 1;
        match arg.to_u64() {
            Some(u) => encode_head(1, u, out),
            None => encode_bignum_tag(3, &arg, out),
        }
    }
}

/// Encode `magnitude` (non-negative) as a CBOR bignum under `tag` (2=pos, 3=neg):
/// tag head, then a byte string of the big-endian minimal magnitude.
fn encode_bignum_tag(tag: u64, magnitude: &BigInt, out: &mut Vec<u8>) {
    encode_head(6, tag, out);
    let bytes = magnitude.magnitude().to_bytes_be();
    encode_head(2, bytes.len() as u64, out);
    out.extend_from_slice(&bytes);
}

fn encode_decimal(d: &BigDecimal, out: &mut Vec<u8>) {
    let (mantissa, scale) = d.normalized().as_bigint_and_exponent();
    let exponent = -scale; // tag 4 value = mantissa * 10^exponent
    encode_head(6, 4, out); // tag 4
    encode_head(4, 2, out); // array of 2
    encode_int(&BigInt::from(exponent), out);
    encode_int(&mantissa, out);
}
```
Note: `BigInt::magnitude()` returns `&BigUint`; `.to_bytes_be()` gives minimal big-endian bytes. Import `num_bigint::BigInt` only; `magnitude()` is inherent.

```rust
//! address `b3:<hex>` (spec §7).

use mangrove_core::Value;

/// The canonical CBOR byte representation (spec §7) of a value.
pub fn canonical_cbor(value: &Value) -> Vec<u8> {
```

```rust
/// Parse `input`, hash it, and assert it equals the trimmed `expected` file.
pub fn run_vector(input: &std::path::Path, expected: &std::path::Path) {
    let src = std::fs::read_to_string(input).unwrap_or_else(|e| panic!("read {input:?}: {e}"));
    let want = std::fs::read_to_string(expected).unwrap_or_else(|e| panic!("read {expected:?}: {e}"));

        .unwrap_or_else(|e| panic!("parse {input:?}: {e}"));

    assert_eq!(got, want.trim(), "hash mismatch for {input:?}");
}
```
